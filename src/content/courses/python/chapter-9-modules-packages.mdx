# Modules and Packages

Modules and packages help organize code into reusable, maintainable units. They allow you to split your code across multiple files and use code written by others.

## Modules

A module is a file containing Python definitions and statements. The file name is the module name with `.py` extension.

### Creating a Module

Create a file `math_utils.py`:

```python
# math_utils.py
def add(a, b):
    return a + b

def multiply(a, b):
    return a * b

PI = 3.14159
```

### Importing Modules

```python
# Import entire module
import math_utils
result = math_utils.add(5, 3)

# Import specific functions
from math_utils import add, multiply
result = add(5, 3)

# Import with alias
import math_utils as mu
result = mu.add(5, 3)

# Import all (not recommended)
from math_utils import *
result = add(5, 3)
```

## Standard Library Modules

Python comes with many built-in modules:

### `math` Module

```python
import math

print(math.pi)           # 3.141592653589793
print(math.sqrt(16))     # 4.0
print(math.ceil(4.2))    # 5
print(math.floor(4.8))   # 4
print(math.pow(2, 3))    # 8.0
```

### `random` Module

```python
import random

# Random integer
number = random.randint(1, 10)

# Random float
float_num = random.random()  # 0.0 to 1.0

# Random choice
fruits = ["apple", "banana", "cherry"]
choice = random.choice(fruits)

# Shuffle list
numbers = [1, 2, 3, 4, 5]
random.shuffle(numbers)
```

### `datetime` Module

```python
from datetime import datetime, timedelta

# Current date and time
now = datetime.now()
print(now)

# Specific date
date = datetime(2024, 1, 15)
print(date)

# Format date
formatted = now.strftime("%Y-%m-%d %H:%M:%S")
print(formatted)

# Time delta
future = now + timedelta(days=7)
print(future)
```

### `os` Module

```python
import os

# Current directory
current_dir = os.getcwd()

# List directory
files = os.listdir(".")

# Check if path exists
exists = os.path.exists("file.txt")

# Create directory
os.makedirs("new_folder", exist_ok=True)

# Environment variables
home = os.environ.get("HOME")
```

### `json` Module

```python
import json

# Convert to JSON string
data = {"name": "Alice", "age": 25}
json_string = json.dumps(data)
print(json_string)  # '{"name": "Alice", "age": 25}'

# Parse JSON string
parsed = json.loads(json_string)
print(parsed["name"])  # "Alice"

# Read from file
with open("data.json", "r") as file:
    data = json.load(file)

# Write to file
with open("output.json", "w") as file:
    json.dump(data, file, indent=2)
```

## Packages

A package is a collection of modules organized in directories.

### Creating a Package

```
my_package/
    __init__.py
    module1.py
    module2.py
    subpackage/
        __init__.py
        module3.py
```

### `__init__.py`

Makes a directory a package (can be empty or contain initialization code):

```python
# my_package/__init__.py
from .module1 import function1
from .module2 import function2

__all__ = ["function1", "function2"]
```

### Importing from Packages

```python
# Import from package
from my_package import module1
from my_package.module1 import function1

# Import from subpackage
from my_package.subpackage import module3

# Import with alias
import my_package.module1 as m1
```

## Installing External Packages

Use `pip` (Python package installer):

```bash
# Install a package
pip install requests

# Install specific version
pip install requests==2.28.0

# List installed packages
pip list

# Uninstall package
pip uninstall requests
```

### Using External Packages

```python
# After: pip install requests
import requests

response = requests.get("https://api.github.com")
data = response.json()
print(data)
```

## Virtual Environments

Isolate project dependencies:

```bash
# Create virtual environment
python -m venv myenv

# Activate (Windows)
myenv\Scripts\activate

# Activate (Mac/Linux)
source myenv/bin/activate

# Deactivate
deactivate
```

## Practice Examples

### Example 1: Custom Calculator Module

```python
# calculator.py
def add(a, b):
    return a + b

def subtract(a, b):
    return a - b

def multiply(a, b):
    return a * b

def divide(a, b):
    if b == 0:
        raise ValueError("Cannot divide by zero")
    return a / b

# main.py
from calculator import add, subtract, multiply, divide

result = add(10, 5)
print(f"10 + 5 = {result}")
```

### Example 2: Configuration Module

```python
# config.py
DATABASE_URL = "localhost:5432"
API_KEY = "secret_key_123"
DEBUG = True

# app.py
import config

print(f"Database: {config.DATABASE_URL}")
if config.DEBUG:
    print("Debug mode enabled")
```

### Example 3: Utility Package

```
utils/
    __init__.py
    string_utils.py
    file_utils.py

# utils/string_utils.py
def reverse_string(s):
    return s[::-1]

def capitalize_words(s):
    return s.title()

# utils/__init__.py
from .string_utils import reverse_string, capitalize_words

# main.py
from utils import reverse_string, capitalize_words

text = "hello world"
print(reverse_string(text))      # "dlrow olleh"
print(capitalize_words(text))    # "Hello World"
```

## Best Practices

1. **Use descriptive module names**
2. **Keep modules focused** (one responsibility)
3. **Document your modules** with docstrings
4. **Use virtual environments** for projects
5. **Create `requirements.txt`** for dependencies:
   ```
   requests==2.28.0
   numpy==1.24.0
   ```
